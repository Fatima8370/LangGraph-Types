# LangGraph-Types
Learning different implementations of LangGraph Agents from the following youtube tutorial : https://youtu.be/jGg_1h0qzaM?si=4exeIahynqYZzHyT

LangGraph is an open-source MIT-licensed extension of LangChain designed for building stateful, multi-actor, and cyclical AI agent workflows. Unlike standard LangChain which executes in linear steps, LangGraph represents the AI workflow as a directed graph where nodes process information and edges determine what happens next.
